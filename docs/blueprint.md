# **App Name**: PromptWala

## Core Features:

- Idea Input: A simple text box to input ideas in Hindi, Hinglish, or English with auto-language detection.
- Prompt Intelligence Engine: Convert casual ideas into high-quality, structured AI prompts by understanding user intent, adding context, defining AI roles, and specifying output formats.
- Prompt Type Selection: Allow users to select the type of prompt they need (e.g., content writing, YouTube script, ad copy).
- Quality Levels: Offer different prompt quality levels (Basic, Advanced, Expert) with varying features and constraints.
- One-Click Copy: Enable users to copy the generated prompt with a single click and provide a 'Use in ChatGPT' button.
- Language Detection and Translation: Automatically detect the language of the user's input and translate the output prompt into Hindi, Hinglish, or English.
- Reasoning Enhanced Prompt Refinement: AI tool leverages the 'Prompt Intelligence Engine' and applies constraints + tone + context based on reasoning for when or if it is needed. 

## Style Guidelines:

- Primary color: Saturated teal (#30BFBF) evokes creativity and technology. It is vibrant and stands out well against the background.
- Background color: Light gray-blue (#E0E5E5), a desaturated version of the primary color for a professional, tech-focused aesthetic. The lightness is suitable for a light scheme.
- Accent color: Blue (#307BBF), a brighter, slightly less saturated color analogous to teal, is chosen for focus.
- Body and headline font: 'Inter' (sans-serif) for a modern, neutral, and readable design.
- Use simple, clear icons to represent different prompt types and features.
- Clean and intuitive layout with a focus on ease of use and quick access to core features.
- Subtle animations and transitions to enhance user experience and provide feedback during prompt generation and copying.
